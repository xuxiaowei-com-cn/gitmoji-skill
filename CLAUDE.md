# Gitmoji Skill

When committing, use the Shortcode column from the [gitmoji.md](gitmoji.md) table as the commit message title prefix.

## Format

- No leading space before the shortcode at the start of a line
- A single space after the shortcode, followed by the commit message body

Format: `:shortcode: your commit message here`

## Data Source

[gitmoji.md](gitmoji.md) — a table with four columns: Emoji, Shortcode, English, 中文
